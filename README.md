# Crypto Bot

Ein automatisierter Trading-Bot für Kryptowährungen.

## Projektstruktur

```
crypto_bot/
├── README.md
```

## Setup

```bash
# Abhängigkeiten installieren
pip install -r requirements.txt

# Bot starten
python main.py
```

## Konfiguration

Konfiguration erfolgt über `.env` oder `config.py`.
